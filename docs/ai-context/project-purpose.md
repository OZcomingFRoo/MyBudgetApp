# MyBudget Project Purpose

MyBudget is an Android application intended to help people manage personal budgeting from a focused mobile interface.

The app should make it easier for users to understand where their money goes, track spending habits, and make practical budgeting decisions without needing a complex finance workflow.

## Current Project State

- Platform: Android
- UI stack: Kotlin with Jetpack Compose and Material 3
- App module: `app`
- Package namespace and application ID: `com.ozcomingfroo.mybudget`
- Current implementation: local-first budgeting app with Compose navigation, dashboard, transaction entry, history, category management, reports, settings, and first-launch onboarding

## Product Direction

MyBudget should evolve into a practical budgeting tool for everyday personal finance. Future AI-assisted work should preserve that direction by favoring clear money tracking flows, simple navigation, and user-facing language that is direct and easy to understand.
