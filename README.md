# MyBudget

MyBudget is a local-first Android budgeting app built with Kotlin, Jetpack
Compose, Room, DataStore, Hilt, and Navigation Compose.

The production Android namespace and application ID are:

```text
com.ozcomingfroo.mybudget
```

Common development commands:

```powershell
.\gradlew.bat assembleDebug
.\gradlew.bat test
```
